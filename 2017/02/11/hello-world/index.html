搭建个人博客

First

下载node

Second

下载msysgit

third

安装hexo

问题
npm ERR! registry error parsing json 错误
可能需要设置npm代理,执行命令
npm config set registry http://registry.cnpmjs.org
hexo:command not found
删除刚刚安装的npm目录，重新执行命令npm install -g hexo安装hexo

fourth

创建文件夹hexo

至此本地博客已搭建好访问localhost:host查看效果

fifth

注册giththub
创建页面仓库格式：yourname.github.io

sixth

生成SSH密钥
ssh-keygen -t rsa -C “你的邮箱地址”，按3个回车，密码为空。
在C:\Users\Administrator.ssh下，得到两个文件id_rsa和id_rsa.pub。
在GitHub上添加SSH密钥
打开id_rsa.pub，复制全文(https://github.com/settings/ssh)Add SSH key，粘贴进去。

seventh

打开hexo文件夹，在config.yml在最后处修改为

注意:后有空格

eighth

编辑文章

在-posts目录下修改，保存后在hexo server预览

ninth

实现hexo的部署

若部署成功可到自己的GitHub Pages查看效果

tips

输入命令hexo deploy出现ERROR Deployer not found:git应该输入命令 npm install hexo-deployer-git–save
出现INFO Deploying: gitINFO Clearing .deploy_git folder…输入 git config user.email”你的邮箱” git config user.name”你的名字
出现”INFO Deploying: gitINFO Clearing .deploy_git folder…INFO Copying files from public folder…输入git config –global core.authorlf false 和hexo clean
输入hexo new page然后照着输邮箱及名字
